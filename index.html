Menentukan Akar-Akar Persamaan Kuadrat

A)Cara Pemfaktoran

Memfaktorkan adalah proses menguraikan suatu bentuk aljabar, seperti persamaan kuadrat, menjadi perkalian dari bentuk-bentuk yang lebih sederhana (faktor-faktornya). Dalam konteks persamaan kuadrat, memfaktorkan berarti mengubah persamaan kuadrat ax2 + bx + c = 0 menjadi bentuk perkalian dua faktor linear, yaitu (x - x1)(x - x2) = 0, di mana x1 dan x2 adalah akar-akar persamaan tersebut. Dengan memfaktorkan, kita dapat menentukan nilai-nilai x yang memenuhi persamaan kuadrat tersebut.

Secara umum, ada tiga kasus utama dalam pemfaktoran persamaan kuadrat agar seluruh bentuk persamaan kuadrat dapat terwakili, yaitu:

  1. Persamaan kuadrat dapat difaktorkan menjadi dua faktor linear dengan koefisien bulat atau rasional
    Contoh: x2 + 5x + 6 = 0 dapat difaktorkan menjadi (x + 2)(x + 3) = 0.
  2. Persamaan kuadrat dapat difaktorkan menjadi dua faktor linear dengan koefisien irasional atau kompleks
    Contoh: x2 + x + 1 = 0 tidak dapat difaktorkan dengan bilangan rasional, tetapi dapat dengan bilangan kompleks.
  3. Persamaan kuadrat tidak dapat difaktorkan (tidak memiliki faktor linear dengan koefisien real)
    Dalam kasus ini, pemfaktoran tidak bisa dilakukan secara sederhana dan harus menggunakan rumus kuadrat atau melibatkan akar-akar irasional/kompleks.

    4. Contoh Soal: Memfaktorkan Persamaan Kuadrat

    Faktorkan persamaan kuadrat berikut dan tentukan akar-akarnya:

    $$3z^{2} + 2z - 85 = 0$$

    Langkah 1: Kalikan koefisien \(a\) dan konstanta \(c\):

    $$a \times c = 3 \times (-85) = -255$$

    Langkah 2: Cari dua bilangan \(m\) dan \(n\) sehingga:

    $$m + n = b = 2$$
    $$m \times n = a \times c = -255$$

    Bilangan yang memenuhi adalah \(m = 17\) dan \(n = -15\).

    Langkah 3: Pecah suku tengah menggunakan \(m\) dan \(n\):

    $$3z^{2} + 17z - 15z - 85 = 0$$

    Langkah 4: Kelompokkan dan faktorkan per pasangan:

    $$(3z^{2} + 17z) + (-15z - 85) = 0$$
    $$z(3z + 17) - 5(3z + 17) = 0$$
    $$(3z + 17)(z - 5) = 0$$

    Langkah 5: Tentukan akar-akar persamaan:

    $$3z + 17 = 0 \Rightarrow z = -\frac{17}{3}$$
    $$z - 5 = 0 \Rightarrow z = 5$$

    Jadi, himpunan penyelesaian persamaan kuadrat tersebut adalah:

    $$\{ z \mid z = -\frac{17}{3}, \quad z = 5 \}$$

    B)Melengkapkan Kuadrat Sempurna

    Persamaan kuadrat dalam x adalah suatu persamaan berbentuk $$ax^2 + bx + c = 0,$$ dengan a, b, dan c bilangan real dan \(a \neq 0\).

    Keterangan: x adalah variabel atau peubah a adalah koefisien \(x^2\) b adalah koefisien \(x\) c adalah konstanta persamaan

    $$ ax^2 + bx + c = 0 \iff x^2 + bx + c = 0 - c $$

    $$ \iff x^2 + bx + \left(\frac{1}{2}b\right)^2 = \left(\frac{1}{2}b\right)^2 - c $$

    $$ \iff \left(x + \frac{1}{2}b\right)^2 = \left(\frac{1}{2}b\right)^2 - c $$

    $$ \iff \left(x + \frac{1}{2}b\right) = \pm \sqrt{\left(\frac{1}{2}b\right)^2 - c}, \quad \text{jika} \ \left(\frac{1}{2}b\right)^2 - c \ge 0 $$

    $$ \iff x = -\frac{1}{2}b \pm \sqrt{\left(\frac{1}{2}b\right)^2 - c}, \quad \text{jika} \ \left(\frac{1}{2}b\right)^2 - c \ge 0 $$
